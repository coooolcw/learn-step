嵌套两个for循环,然后在最内部判断是否能被最后一个整除  
```
for(let i = 1; i <= Number.floor(n / 5); i++){
  for(let j = 0; j <= Number.floor((n - i * 5)/3; j++) {
    if((n - i * 5 - j * 3) % 2 === 0) {
      ansArr.push({
        5 : i,
        3 : j,
        2 : (n - i * 5 - j * 3) / 2
      })
    }
  }
}
```
